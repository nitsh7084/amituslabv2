Mafia
Project Description
Mafia is an intelligent multiplayer game about two group of players: Mafias and Citizens, and their helper players, the game ends when one of the group of players dies.
What we did
Everything
Mafia is an intelligent multiplayer game that is played by 8 players at a time, the players include God,Detective,Dacoit,Healer, a group of 3 citizens and a group of 3 mafias. We were responsible for the design, branding as well as the development of the game.
Goqii
Project Description
Goqii and Touchpoints Solutions Inc(a helper consulting firm) chose us to help them develop their Coach Application
What we did
Web Design for the coach application, integration of the application with the application backend.
Goqii is a fitness application that helps users with assigning them their personal coaches to keep track of their fitness quotient, the coaches guide the users at every step of the day in maintaining their health and making sure that the user is motivated for their work. We helped goqii design their coach application and integrating the application design with the backend of the device.
SyncAccounts
Project Description
Syncaccounts is an application that allows you to manage all your social accounts from one place, it provides the user with a dashboard for running analytics on their social media accounts.
What we did
Everything
Syncaccounts is an application that allows users to sync their social media accounts and then run an analytics on them through which users can identify the trends within their accounts. Syncaccounts is an in-house project and therefore we took care of the complete application.
UFaber
Project Description
We worked with UFaber in August 2015, they chose us for the redesign of the core parts of their application.
What we did
Web design of the core pages.
UFaber is an online education platform which allows users to pick their subjects and courses and learn new things everyday, it offers free as well as premium courses. We designed the core pages for the UFaber website including the homepage, the courses page and the individual course page.
